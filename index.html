Deploying Python Apps

Topic: Converting .PY to .EXE

In this lesson, we learn how to take our Python logic and turn it into a standalone computer application.

Why Convert to .EXE?

  • Portability: Run your app on computers that don't have Python installed.
  • Protection: Users can't accidentally change your code.
  • Professionalism: It feels like a real product, not just a script.
  • One-Click: Simply double-click to start!

The Tool: PyInstaller

We use a library called PyInstaller to bundle our code.

To install, run this command in your terminal:

pip install pyinstaller

Step 1: Prep Your Code

Make sure your Grading App script is ready:

score = float(input("Average: "))
if score >= 90: print("A")
# ... rest of your logic ...

input("Press Enter to Exit")

Crucial: Add that last line so the window doesn't close immediately!

Step 2: Conversion

  1. Open Command Prompt (CMD).
  2. Go to your folder: cd MyProject
  3. Run the command:
pyinstaller --onefile grader.py

The --onefile flag ensures everything is packed into a single .exe.

Step 3: Where is the App?

Once finished, look at your project folder. You will see a new folder named:

📂 dist

Your grader.exe is inside this folder! You can now send this file to your friends.

Pro Tip: Custom Icons

Want your app to look unique on the desktop?

pyinstaller --onefile --icon=myicon.ico grader.py
  • Find an .ico file online.
  • Use the command above to attach it to your application.
